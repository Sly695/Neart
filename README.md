# Neart

Les prérequis :  

- GIT installé
- NODE.JS installé
- NPM installé
- EXPO GO installé sur son téléphone

Comment lancer l'app : 

- Après avoir clone le projet :
- Se placer dans la racine du projet
- lancer npm install
- lancer "npx expo start" ou "npm start"
- scanner le qr code

- L'app se lance
